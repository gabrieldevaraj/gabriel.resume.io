Global Liferay Portal Management

Project Overview

Managed and enhanced global-level client portals using Liferay DXP framework at Accenture Technologies. Developed custom Java components, JSP pages, and FreeMarker templates to improve portal functionality, user experience, and performance. Worked closely with international teams to deliver scalable solutions that served thousands of users across multiple regions.

Key Responsibilities

  • Developed and maintained custom portlets using Java, JSP, and FTL for portal customization
  • Managed MySQL database operations and optimized queries for better performance
  • Implemented version control using Git/GitHub for collaborative development
  • Followed Agile methodologies, working in sprints with close team collaboration
  • Utilized JIRA for project management, issue tracking, and sprint planning
  • Enhanced portal features including user management, content management, and workflow automation

Technologies Used

Liferay DXP Java JSP FTL MySQL Git/GitHub Eclipse IDE JIRA

Project Impact

Successfully maintained and enhanced global portals serving thousands of users across multiple countries. The improvements resulted in better user experience, reduced load times, and increased portal reliability. Collaborated effectively with cross-functional teams to deliver high-quality solutions within tight deadlines.
